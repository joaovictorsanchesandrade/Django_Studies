# Templates no Django

Os **templates** são responsáveis pela camada de **apresentação** da aplicação Django. Eles definem como os dados enviados pelas *views* serão exibidos ao usuário, garantindo a separação entre **lógica** e **interface**.

O Django utiliza o **Django Template Language (DTL)**, uma linguagem própria para renderização de HTML dinâmico de forma segura, organizada e eficiente.

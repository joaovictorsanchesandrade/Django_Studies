# Generic Views

As **Generic Views** do Django são ideais quando você precisa **manipular models de forma rápida e padronizada**, sem escrever views complexas ou repetir lógica desnecessária.

Elas fornecem implementações prontas para operações comuns, permitindo criar um **CRUD completo** com menos código, mais organização e maior legibilidade.

## Principais Generic Views

* **[ListView](ListView.md)**
  Exibe uma lista de objetos de um model.

* **[DetailView](DetailView.md)**
  Mostra os detalhes de um único objeto.

* **[CreateView](CreateView.md)**
  Responsável pela criação de novos registros.

* **[UpdateView](UpdateView.md)**
  Permite a edição de registros existentes.

* **[DeleteView](DeleteView.md)**
  Gerencia a remoção de registros.
